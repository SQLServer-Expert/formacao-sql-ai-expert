Bem-vindo ao repositório oficial da formação **SQL AI Expert**.

- Linguagem SQL
- Desenvolvimento de Banco de Dados
- Inteligência Artificial aplicada ao SQL Server

- Microsoft Foundry
- Preparação para Certificação Microsoft DP-800
